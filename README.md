# 🌦️ Weather Forecasting System

A Full Stack Web Application that provides real-time weather updates using API integration, frontend UI, backend service, and database storage.

---

## 🚀 Features

- 🌍 Search weather by city name  
- 🌡️ Shows current temperature  
- 🌧️ Weather conditions (rain, clear, cloudy etc.)  
- 💨 Wind speed details   
- 📱 Responsive UI design  

---

---

### Frontend

- JavaScript  

### Backend
- Java (Spring Boot)
